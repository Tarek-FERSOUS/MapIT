\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
% TODO: Replace only if your supervisor requests a different abstract wording.
MapIT is a web-based platform designed to centralize IT infrastructure documentation and incident knowledge management in a single, secure workspace. The project addresses the common operational problem of fragmented notes, duplicated troubleshooting, and weak traceability between incidents and the knowledge used to resolve them. The proposed solution combines a structured data model, role-based access control, and a clear user interface to support incident registration, document maintenance, and knowledge reuse. The system is implemented as a full-stack application with a separated frontend, backend API, and relational database, making it suitable for teams that need a lightweight but disciplined documentation environment.

\textbf{Keywords:} IT infrastructure, incident management, knowledge management, documentation platform, role-based access control.
